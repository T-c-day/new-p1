# Employee Reimbursement App

## Project Description

The Employee Reimbursement App allows an employee to submit an expense reimbursement and for their manger to view and accept/deny the expense report. 

## Technologies Used

* JDBC 
* Java 
* PostgreSQL 
* JavaScript 
* CSS 
* HTML 
* Servlets 
* JUnit

## Features
